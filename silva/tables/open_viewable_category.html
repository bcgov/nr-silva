open_viewable_category

-1 rows


Description

OPEN_VIEWABLE_CATEGORY identifies the open categories that can be accessed by all licensees using the application. The system must use this stored information as a means to allow access for all users to openings that have an opening category with this designation enabled or openings in a Free Growing status. All other opening categories will continue to be restricted based on the current business rules.
